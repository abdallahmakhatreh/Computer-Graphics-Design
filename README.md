# Computer Graphics Design Portfolio

A curated collection of digital design, vector illustration, and visual composition assets developed during my undergraduate studies at Yarmouk University. This repository showcases my baseline skills in layout design, layer architecture, branding, and asset composition using industry-standard tools like Adobe Photoshop and CorelDRAW.

---

## 🛠️ Software & Tools Used
* **Adobe Photoshop:** Advanced raster manipulation, photo-compositing, and non-destructive image editing.
* **CorelDRAW:** Mathematical vector path illustration and scalable graphic design.

---

## 📁 Core Portfolio Projects

### 1. Automotive Media Matte Design (`Abdallah_A2.psd`)
A high-fidelity photo-composition that blends multiple distinct automotive elements into a single, cohesive banner. This project highlights my understanding of light, shadows, and spatial layout:
* **Layer Architecture:** Built using a non-destructive workflow with smart objects, custom clipping masks, and adjustment layers.
* **Lighting & Reflections:** Manually adjusted highlights and surface textures to ensure the multi-car placement blends naturally into the custom environment backdrop.
* **📥 Project Source File:** Due to GitHub's file size limits, the raw master layered file is securely hosted in the project release. You can download the full 27.4 MB asset here: **[Download Abdallah_A2.psd](https://github.com/abdallahmakhatreh/Computer-Graphics-Design/releases/download/v1.0.0/Abdallah_A2.psd)**

### 2. Corporate Branding Identity Layout (`Assignment 1.png`)
A commercial graphic designed for a fictional business named "Abood Programming Company." 
* Focuses on essential print and digital design principles, including typographic weight hierarchy and intentional alignment.
* Uses balanced empty space (negative space) to create a clean, modern, and easily readable corporate identity template.

### 3. Infinite Scalability Vector Illustration (`Abdallah_A1.cdr`)
A geometric vector blueprint built entirely using mathematical lines and anchors within CorelDRAW. 
* Because it is built using mathematical nodes instead of static pixels, the final artwork can scale to any resolution or print size without ever losing its crisp quality.
